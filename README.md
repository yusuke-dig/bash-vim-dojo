# Bash & Vim Dojo

毎日15分、VimとBashを組み合わせて学ぶ練習プロジェクト。

## 使い方

```bash
# レッスンを読む
vim lessons/day01/lesson.md

# 演習を開く（hjkl移動・矢印キー禁止！）
vim lessons/day01/exercise.sh

# 採点
bash lessons/day01/test.sh
```
