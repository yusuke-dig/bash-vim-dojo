# Bash & Vim Dojo — 30日カリキュラム

## Phase 1: 基礎固め（Day 1〜10）

| Day | Vimテーマ | Bashテーマ | 組み合わせゴール |
|-----|-----------|------------|----------------|
| 01 | 移動（hjkl, w, b, 0, $） | 変数・echo | スクリプトをVimで書いて実行 |
| 02 | 挿入・削除（i, a, x, dd） | if文・比較演算子 | 条件分岐スクリプトをVimで編集 |
| 03 | 検索（/, n, N, *） | for文・seq | ループスクリプト内をVim検索で移動 |
| 04 | 置換（:s/old/new/g） | while文・read | `:s`でスクリプト内変数をリネーム |
| 05 | ヤンク・ペースト（y, p, yy） | 関数（function） | 関数をコピーして再利用 |
| 06 | ビジュアルモード（v, V, Ctrl-v） | 配列 | 複数行を選択して一括編集 |
| 07 | ウィンドウ分割（:sp, :vsp） | パイプ・リダイレクト | 分割画面でスクリプトと出力を確認 |
| 08 | マクロ（q, @） | grep・正規表現 | マクロで繰り返しコードを生成 |
| 09 | :! でシェル実行 | ファイル操作（cp, mv, find） | VimからBashを直接実行 |
| 10 | 復習Day | 復習Day | Phase 1の総合演習 |

## Phase 3: Bash 実践（Day 21〜38）

> Day 21 以降は Vim テーマなし。Vim は「スクリプトを書くエディタ」として使うだけ。lesson.md は Bash のチートシートとコード例に集中し、exercise.sh は5問すべて Bash。

### エラーハンドリング編（Day 21〜24）

| Day | Bashテーマ | 学習ゴール |
|-----|-----------|-----------|
| 21 | 終了コードとエラーハンドリング（$? / set -e / set -o pipefail） | 堅牢なCIスクリプトを書く |
| 22 | 条件実行パターン（&& / \|\| / if cmd; then） | CIでよく見るワンライナーを読む・書く |
| 23 | 環境変数とデフォルト値（${VAR:-default} / ${VAR:?error}） | CI環境変数を安全に扱う |
| 24 | 🥋 Bash道場①: sed + awk + xargs 反復練習（組み合わせ） | Day 11・13・14 の実力を組み合わせで試す |

### 文字列・変数操作編（Day 25〜28）

| Day | Bashテーマ | 学習ゴール |
|-----|-----------|-----------|

| 28 | 🥋 Bash道場②: sort / uniq / cut / paste 練習 | テキスト集計・整形の定番コマンドを体で覚える |
