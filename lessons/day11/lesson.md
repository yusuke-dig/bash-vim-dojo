## Vim チートシート（今日覚える4コマンド）

| コマンド | 動作 | 例 |
|---------|------|----|
| `ma` | カーソル位置にマーク `a` をセット | （aはa〜zの任意の文字） |
| `'a` | マーク `a` の行頭へジャンプ | |
| `` `a `` | マーク `a` の正確な位置へジャンプ | （行頭ではなく列も保持） |
| `''` | 直前のジャンプ元へ戻る | 行き来に便利 |

### よく使うパターン

```
# 作業中の2箇所を行き来する
ma          → 場所Aにマークをセット
（別の場所へ移動して編集）
'a          → 場所Aの行頭へ戻る

# 範囲選択との組み合わせ
ma          → 起点にマーク
（数行下へ移動）
d`a         → マーク位置まで削除
y`a         → マーク位置までヤンク
```

## Bash チートシート（今日覚える4コマンド）

| コマンド | 動作 | 例 |
|---------|------|----|
| `sed 's/old/new/' file` | 各行の最初のマッチを置換 | `sed 's/foo/bar/' data.txt` |
| `sed 's/old/new/g' file` | 各行の全マッチを置換（g フラグ） | `sed 's/foo/bar/g' data.txt` |
| `sed -n '2,4p' file` | 2〜4行目だけ表示（-n で自動出力を抑制） | `sed -n '1p' data.txt` |
| `sed '/パターン/d' file` | パターンにマッチする行を削除 | `sed '/^#/d' script.sh` |

### コード例（そのまま使える）

```bash
# "apple" を "orange" に置換して表示
sed 's/apple/orange/' fruits.txt

# ファイル全体で全マッチを置換して別ファイルに保存
sed 's/hello/world/g' input.txt > output.txt

# 1行目だけ抽出
sed -n '1p' data.txt

# コメント行（# で始まる行）を除いて表示
sed '/^#/d' script.sh

# 複数の置換を重ねる（-e で連結）
sed -e 's/foo/bar/g' -e 's/baz/qux/g' data.txt
```

## 組み合わせポイント（なぜ一緒に学ぶか）

Vim のマークと sed は「どこを編集するか」と「どう変換するか」をそれぞれ担う。

```
# 典型的な使い方
vim data.txt
  → ma で編集開始箇所をマーク
  → :!sed 's/old/new/g' data.txt でプレビュー
  → 結果を確認して 'a で戻り、手動で修正

# Vim の :s と sed の使い分け
:s/old/new/g     → Vim バッファ内を置換（保存前）
sed 's/old/new/g' file → 外部ファイルを加工してから取り込む
```
