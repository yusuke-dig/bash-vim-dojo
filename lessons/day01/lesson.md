# Day 01: Vim移動 × Bash変数・echo

## 今日のゴール（15分で達成）

`exercise.sh` をVimで開き、変数とechoを使った小さなスクリプトを完成させる。
Vimのhjkl移動だけでカーソルを動かす（矢印キー禁止！）。

---

## Vim チートシート

| コマンド | 動作 |
|---------|------|
| `h` | 左に1文字 |
| `l` | 右に1文字 |
| `j` | 下に1行 |
| `k` | 上に1行 |
| `w` | 次の単語の先頭へ |
| `b` | 前の単語の先頭へ |
| `0` | 行頭へ |
| `$` | 行末へ |
| `gg` | ファイルの先頭へ |
| `G` | ファイルの末尾へ |
| `i` | カーソル位置から挿入モード |
| `Esc` | ノーマルモードに戻る |
| `:w` | 保存 |
| `:q` | 終了 |
| `:wq` | 保存して終了 |

**今日の鉄則**: hjklで移動できないと感じたら `0` や `$` `w` `b` を組み合わせる。

---

## Bash チートシート

```bash
# 変数への代入（スペースなし！）
name="Alice"
number=42

# 変数の参照（$をつける）
echo $name
echo "Hello, $name!"

# コマンドの結果を変数に代入
today=$(date +%Y-%m-%d)
echo "今日は $today"

# 数値計算
result=$((3 + 5))
echo $result
```

**今日の鉄則**: `name = "Alice"` はエラー。`=` の前後にスペースを入れない。

---

## 組み合わせポイント

Bashスクリプトを書く = Vimで `.sh` ファイルを編集すること。
今日から「Bashを書く = Vimを使う」を習慣にする。

```
vim exercise.sh   # Vimでスクリプトを開く
```

ノーマルモードで `:!bash %` と入力すると、
今開いているファイルをその場で実行できる（後日詳しく学ぶ）。
