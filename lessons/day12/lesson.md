# Day 12: 複数ファイル操作（:e, :bn, :bp） × 複数ファイル処理

## 今日のゴール（15分で達成）

Vim のバッファ機能で複数ファイルを行き来しながら、Bash の for ループと glob でまとめてファイルを処理できるようになる。

---

## Vim チートシート（今日覚える5コマンド）

| コマンド | 動作 | 例 |
|---------|------|----|
| `:e ファイル名` | 別ファイルをバッファに開く | `:e source_b.txt` |
| `:ls` | 開いているバッファ一覧を表示 | |
| `:bn` | 次のバッファへ切り替え | |
| `:bp` | 前のバッファへ切り替え | |
| `:b N` | N番目のバッファへ直接切り替え | `:b 2` |

### よく使うパターン

```
# 複数ファイルを開いて行き来する
:e source_a.txt      → 1つ目のファイルをバッファに開く
:e source_b.txt      → 2つ目のファイルをバッファに追加
:ls                  → バッファ一覧で番号を確認
:bn                  → 次のファイルへ
:bp                  → 前のファイルへ
:b 1                 → 1番バッファに直接ジャンプ

# Vim を複数ファイル付きで起動する
vim source_a.txt source_b.txt source_c.txt
```

## Bash チートシート（今日覚える5コマンド）

| コマンド | 動作 | 例 |
|---------|------|----|
| `cat f1 f2 f3 > out` | 複数ファイルを連結して保存 | `cat a.txt b.txt > all.txt` |
| `for f in *.txt; do ...; done` | glob でファイルを順番に処理 | |
| `grep -h pattern *.txt` | 複数ファイルを検索（ファイル名プレフィックスなし） | |
| `head -1 "$f"` | ファイルの先頭1行を取得 | |
| `tail -1 "$f"` | ファイルの末尾1行を取得 | |

### コード例（そのまま使える）

```bash
# 3つのファイルを連結して1つに保存
cat source_a.txt source_b.txt source_c.txt > result.txt

# glob で全 .txt ファイルを順番に処理
for f in source_*.txt; do
  echo "Processing: $f"
done

# 複数ファイルから "e" を含む行を検索（ファイル名なし）
grep -h "e" source_*.txt > result.txt

# 各ファイルの先頭行を収集して1つのファイルに追記
for f in source_*.txt; do
  head -1 "$f" >> result.txt
done

# 各ファイルの行数を "ファイル名: N lines" 形式で出力
for f in source_*.txt; do
  count=$(wc -l < "$f" | xargs)
  echo "$f: $count lines"
done
```

## 組み合わせポイント（なぜ一緒に学ぶか）

「複数ファイルをVimで行き来する」と「複数ファイルをBashでまとめて処理する」は、同じ「複数ファイルを扱う」作業の表と裏。

```
# 実際の作業イメージ
vim source_a.txt source_b.txt source_c.txt
  :ls               → バッファ一覧を確認
  :bn / :bp         → ファイルを行き来して中身確認
  :!bash exercise.sh → Bashで一括処理を実行して結果確認
  :b 2              → 気になったファイルに直接戻る
```
