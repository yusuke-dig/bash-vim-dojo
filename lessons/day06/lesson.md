# Day 06: Vimビジュアルモード（v, V, Ctrl-v） × Bash配列

## 今日のゴール（15分で達成）

- Vimの `V` で複数行を選択してヤンク・削除・インデントできる
- `Ctrl-v` で矩形選択し、複数行を一括編集できる
- Bashで配列を定義・参照・ループ処理できる
- 配列要素が縦並びのコードを `Ctrl-v` で一括変更できる

## Vim チートシート（今日覚える5コマンド）

| コマンド | 動作 |
|----------|------|
| `v` | 文字単位ビジュアルモード（1文字ずつ選択） |
| `V` | 行単位ビジュアルモード（1行まるごと選択） |
| `Ctrl-v` | 矩形ビジュアルモード（複数行を列で選択） |
| `Ctrl-v` + `I` + 文字 + `Esc` | 矩形選択した全行の先頭に一括挿入 |
| `gv` | 直前のビジュアル選択を再選択 |

> **矩形選択（Ctrl-v）** は縦方向に「列」を選ぶ。配列要素や変数名が縦に並ぶとき最強。

## Bash チートシート（今日覚える5コマンド）

| 書き方 | 動作 |
|--------|------|
| `arr=(a b c)` | 配列の定義（スペース区切り） |
| `${arr[0]}` | インデックスで1要素を取得（0始まり） |
| `${arr[@]}` | 全要素を展開 |
| `${#arr[@]}` | 配列の要素数 |
| `arr+=(d)` | 配列に要素を追加 |

## 組み合わせポイント（なぜ一緒に学ぶか）

Bashの配列は要素が縦に並ぶことが多い。そこで `Ctrl-v` の出番。

```
# 例: fruits 配列の各行先頭に "  - " を一括で追加したい
fruits=(
  apple
  banana
  cherry
)

# Ctrl-v で apple〜cherry の行頭を矩形選択
# I で挿入モード → "  - " と打って Esc
# → 全行に "  - " が挿入される
```

`V` での行選択も便利。不要な要素を `Vd` で複数行ごと削除、`yy` 済みの行を `p` で追加する作業が速くなる。
